🔢 Customer Account Setup

Create account credentials for approved customers so they can log in to their dashboard

📋 How This Works — Step by Step:

  1. Fill in the customer's details below
  2. Set their account number (use Account Generator in Admin) and password
  3. Click "Create Account Credentials"
  4. Copy the credentials shown
  5. Open the login.html file and add the customer to the database section
  6. Send the customer their account number and password by email
  7. Customer logs in at discoverjp.org/login.html

📋 How to Add Customer to Login System

After creating credentials, you need to add the customer to login.html on GitHub:

  1. Go to GitHub → discover-bank-management
  2. Click on login.html
  3. Click edit ✏️ → "In place"
  4. Find the line: const customers = {
  5. Paste the generated code snippet after the last customer entry
  6. Click "Commit changes"
  7. Wait 2 minutes then customer can login! ✅
